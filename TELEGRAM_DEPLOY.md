# Как поставить мини-приложение в Telegram

## 1. Собрать проект

```bash
npm run build
```

Готовые файлы появятся в папке `dist/`.

---

## 2. Опубликовать в интернете (HTTPS)

Мини-приложение должно открываться по **HTTPS**. Варианты:

### Вариант A: Vercel (удобно и бесплатно)

1. Зарегистрируйся на [vercel.com](https://vercel.com).
2. Подключи репозиторий с проектом (GitHub/GitLab) или залей папку `dist` через Vercel CLI.
3. Команда для деплоя из папки проекта:
   ```bash
   npx vercel --prod
   ```
   Либо: залей проект на GitHub → в Vercel нажми "Import Project" → укажи корень и команду сборки `npm run build`, папку вывода `dist`.
4. После деплоя получишь ссылку вида: `https://твой-проект.vercel.app`.

### Вариант B: Netlify

1. Зарегистрируйся на [netlify.com](https://netlify.com).
2. "Add new site" → "Deploy manually" или подключи Git.
3. Для ручного деплоя: перетащи папку `dist` в окно Netlify.
4. Ссылка будет вида: `https://твой-сайт.netlify.app`.

### Вариант C: Свой хостинг

Залий содержимое папки `dist/` на любой хостинг с HTTPS (например, хостинг с SSL или свой сервер с nginx + Let's Encrypt).

---

## 3. Создать бота в Telegram

1. Открой [@BotFather](https://t.me/BotFather) в Telegram.
2. Отправь команду: `/newbot`.
3. Введи имя бота (например: Crash Bunny).
4. Введи username бота (должен заканчиваться на `bot`, например: `crash_bunny_bot`).
5. BotFather пришлёт **токен бота** — сохрани его (он понадобится для серверной части, если будешь делать её позже).

---

## 4. Привязать мини-приложение к боту

1. В [@BotFather](https://t.me/BotFather) отправь: `/mybots`.
2. Выбери своего бота.
3. Нажми **"Bot Settings"** → **"Menu Button"** (или **"Configure"** у пункта про Web App).
4. Выбери **"Configure menu button"** (или **"Edit menu button URL"**).
5. Введи **URL твоего приложения** — ту самую ссылку с Vercel/Netlify/хостинга, например:
   ```
   https://твой-проект.vercel.app
   ```
6. При необходимости задай текст кнопки в меню (например: «Играть»).

Либо через Bot API (если знаешь, как слать запросы):

- Метод [setChatMenuButton](https://core.telegram.org/bots/api#setchatmenubutton): в `menu_button` укажи тип `web_app` и твой URL.

---

## 5. Открыть мини-приложение

- Зайди в бота в Telegram.
- Нажми на кнопку **меню** (слева от поля ввода) — там будет твоя кнопка с ссылкой на приложение.
- Либо отправь боту команду, которая открывает Web App (если настроил так в коде бота).

Ссылка на приложение в формате Telegram:

```
https://t.me/ИМЯ_ТВОЕГО_БОТА/app
```

(Работает, если у бота настроена кнопка меню с этим URL.)

---

## Краткий чеклист

1. [ ] `npm run build` — собрал проект.
2. [ ] Выложил содержимое `dist/` на хостинг с HTTPS.
3. [ ] Создал бота через @BotFather.
4. [ ] В настройках бота указал URL мини-приложения (Menu Button / Web App URL).
5. [ ] Открыл бота в Telegram и нажал на кнопку меню — открылось приложение.

После этого мини-приложение будет доступно в Telegram по кнопке в меню бота.
